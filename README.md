# Zielony Kąt - Sklep z roślinami

## Opis aplikacji
Aplikacja webowa pełniąca rolę katalogu sklepu z roślinami doniczkowymi.

## Wykorzystane technologie
Python, Django, SQLite, Bootstrap 5.

## Wkorzystanie AI
Wykorzystanie Gemini, Copilot. 
* **Samodzielnie:** Konfiguracja VS Code, zarządzanie terminalem, środowisko wirtualne, obsługa panelu admina, naprawa błędów ścieżek, commitowanie.
* **Z pomocą AI:** Generowanie boilerplate'u kodu Django, pisanie testów jednostkowych.
